# Cats-vs-Dogs_image-Classifier
A Conv-net which is trained using Keras(uses Tensorflow in backend) to classify between Cats and Dogs.

## Modules Used
- Keras
- scikit-learn
- numpy
- pandas
- opencv
- matplotlib
- tqdm
